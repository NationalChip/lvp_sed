最近修改
========================

相比v1.0.0-rc2，本版本修复或实现了以下的问题：

1、G-sensor驱动更新。
2、更新G-sensor vad算法。
3、优化audio in和audio out部分的调用，支持amic、dmic同时使用，支持mic数量选择。
4、部分代码更新版权头信息。
5、更新模型自动部署脚本。
6、自适应调整vad参数。
7、更新bootx。
