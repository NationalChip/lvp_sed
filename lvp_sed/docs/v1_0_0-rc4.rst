最近修改
========================

相比v1.0.0-rc3，本版本修复或实现了以下的问题：

1. 英文通用模型更新到 v0.1.8，此版本不需要使能 NPU Run In Flash
2. 添加 beamsearch + lm
3. 增加feed模式
4. 优化G-sensor接口存取数据方式，提高速度
5. 调整G-vad算法实现和部分默认阈值，优化了体验效果
6. 同步、验证scpu更新i2c接口，面向字节的接口
